Cliente X Server

Cliente

O cliente é como você, o usuário final, que usa um dispositivo (como um computador, celular ou tablet) para fazer coisas como acessar sites, enviar e-mails ou baixar arquivos. O cliente envia solicitações para o servidor, que é como um "computador poderoso" que guarda informações e responde às solicitações dos clientes. Alguns exemplos de clientes são os navegadores da web (como o Google Chrome ou o Mozilla Firefox), aplicativos de e-mail (como o Gmail) e programas de mensagens (como o WhatsApp).

Servidor

O servidor é como um "computador poderoso" que armazena informações e está sempre ligado para atender às solicitações dos clientes. Ele recebe as solicitações dos clientes, processa as informações necessárias e envia as respostas de volta. Os servidores podem ser usados para hospedar sites, enviar e receber e-mails, armazenar dados em bancos de dados, compartilhar arquivos e muito mais. Exemplos de servidores incluem os servidores web, servidores de e-mail, servidores de banco de dados e servidores de arquivos.

Tipografia

A tag <h1> a <h6> é usada para criar títulos de diferentes tamanhos. O <h1> é o maior e o <h6> é o menor. Você pode usar essas tags para destacar partes importantes do seu texto.

A tag <p> é usada para criar parágrafos. É como uma quebra de linha onde você pode colocar o texto do seu documento.

A tag <strong> é usada para destacar texto em negrito. Se você quiser que uma parte do seu texto fique em destaque, você pode envolvê-la com essas tags.

A tag <i> é usada para enfatizar o texto, geralmente exibindo-o em itálico. Se você quiser dar ênfase a uma parte do seu texto, você pode usar essas tags.

A tag <u> é usada para sublinhar o texto. Se você quiser enfatizar ainda mais uma parte do texto, pode envolvê-la com essa tag.

A tag <s> é usada para mostrar o texto com um risco no meio, como se estivesse riscado. Isso pode ser útil se você quiser indicar que uma parte do texto está obsoleta ou não é mais válida.

Imagem

  1. Escolha a foto que deseja adicionar ao seu site HTML.
  2. Salve a foto em um local acessível ou copie seu endereço.
  3. Use a tag <img> para inserir a foto no seu documento HTML. A tag <img> não tem uma tag de fechamento, então você pode fechá-la imediatamente.
  4. Salve seu arquivo HTML e abra-o em um navegador para visualizar a foto.

Tenha em mente que é importante fornecer o caminho correto para a foto no atributo src. Se a foto não estiver sendo exibida corretamente, verifique se o caminho está correto e se a foto está no local esperado.

Você também pode adicionar atributos adicionais à tag <img>, como width e height, para especificar as dimensões da imagem, ou class e id para estilizar a foto usando CSS.

Ordenar

<ol> (ordered list): É a tag usada para criar listas ordenadas, onde os itens são numerados automaticamente pelo navegador. Cada item da lista é definido pela tag <li> (list item). Você pode usar a tag <ol> para criar listas numeradas, como listas de passos ou hierarquias.

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

<ul> (unordered list): É a tag usada para criar listas não ordenadas, onde os itens não são numerados. Assim como a tag <ol>, cada item da lista é definido pela tag <li>. A lista é representada por marcadores, geralmente pontos ou círculos, mas o estilo visual dos marcadores pode ser personalizado com CSS.

Exemplo:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
